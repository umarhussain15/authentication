Master UI


About this page

This page contains some examples that you can use for your modules. The goal is to create an interface that is consistent as well beautiful.
Created by Shabana Akhtar Baig (04879) and Asfandyar Hashmi (05312).


How to use Master UI

Link our master.css stylesheet file in your <head>

Use Inspect Element to view the CSS class and copy it so you can use it in your own modules.

Buttons use e.g. class="btn btn-default"



Forms use class="form-control" for inputs



Navbar use class="navbar navbar-default" for nav
